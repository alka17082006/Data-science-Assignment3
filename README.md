# 🌍 Global Terrorism Data Analysis

## 📌 Project Overview
This project analyzes global terrorism data from 1970 to 2017 using data analysis techniques. The aim is to identify patterns, trends, and key insights related to terrorist activities across the world.

---

## 🎯 Objectives
- Analyze terrorism trends over the years
- Identify most affected countries and regions
- Study different types of attacks
- Understand weapons used in attacks
- Analyze casualties and impact

---

## 📊 Dataset Information
- Dataset: Global Terrorism Database (1970–2017)
- Total Records: 180,000+ incidents
- Features include:
  - Year of attack
  - Country and region
  - Attack type
  - Weapon type
  - Number of people killed and wounded

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📈 Visualizations
The project includes:
- Year-wise attack trend (Line Chart)
- Top affected countries (Bar Chart)
- Attack types distribution (Pie Chart)
- Weapons used (Bar Chart)
- Casualties distribution (Histogram)
- Region-wise analysis

---

## 🔍 Key Insights
- Terrorist activities increased significantly after 2000
- Bombing/Explosion is the most common attack type
- Explosives are the most frequently used weapons
- A few countries account for most attacks
- Most attacks cause low casualties, but some are extremely deadly

The analysis highlights important global terrorism trends and provides meaningful insights that can help improve security measures and decision-making.
